Browser support

MQ is powered by HTML5 <audio> tag which means you'll need one of the modern browsers that support HTML5:

Internet Explorer * Chrome Opera Safari ** Firefox Android iOS
HTML5 Support 9.0+ 3.0+ 10.5+ 3.1+ 3.5+ 2.2+ 3.0+
MQ tested 9.0+ 35.0 29.0+

*Internet Explorer users must add an exception to the Privacy settings to allow using cookies for the "google.com" domain: 1) Go to Tools->Internet Options->Privacy->Sites 2) Enter "google.com" and press "Allow" button.

**Safari users must have Quick Time installed to enable HTML5 <audio> feature.

Please, also note, to use MQ the Javascript must be enabled in your browser.


Shortcuts

x Play
c Play/pause
mum 8 Volume up
mum 2 Volume down
num 4, ← Play previous track
num 6, → Play next track
del Delete selected tracks from playlist
m Mute
r Repeat
s Shuffle
